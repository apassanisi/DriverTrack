### Usage

| Use                    | Command           | URL                                            |
| ---------------------- | ----------------- | ---------------------------------------------- |
| Install locally        | `nvm use && yarn` |                                                |
| Start local dev server | `yarn dev`        | [http://localhost:3000](http://localhost:3000) |
| Static build           | `yarn build`      |                                                |
| Preview static build   | `yarn serve`      | [http://localhost:4173](http://localhost:4173) |
